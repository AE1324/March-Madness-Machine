## Dashboard Screenshots

This repo’s UI is a Streamlit app. To document the research workflow publicly:

1. Generate a modest number of brackets (e.g. `100,000`) and load them in the app.
2. Open the following screens and capture screenshots:
   - **Stats → Perfect bracket survival** (curve)
   - **Stats → Pick percentages by round**
   - **Admin → Recompute pick percentages and brackets at risk** (progress UI)
3. Save screenshots into this repository and reference them from `README.md`.

Suggested naming:
- `docs/screenshots/perfect_brackets.png`
- `docs/screenshots/pick_percentages.png`
- `docs/screenshots/recompute_progress.png`

If you don’t want to include images, you can omit them and keep the text-only references in this guide.
